# Review Process (Codex)

Codex reviews claude-code's pushed changes and decides accept or reject.

## Review Checklist

- Matches the plan scope and steps
- Acceptance criteria met
- Tests added/updated and passing (or rationale noted)
- No unintended regressions
- Design specs followed (if applicable)
- Documentation updated as required

## Outcomes

- Accept: Codex commits the changes with a clear message.
- Reject: Codex documents blockers and issues a revised plan.
