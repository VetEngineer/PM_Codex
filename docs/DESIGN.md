# Design Process (gemini-cli)

gemini-cli provides design guidance and artifacts for the implementation.

## Deliverables

- UI/UX direction and visual language
- Component specs or wireframes (if needed)
- Style tokens (colors, typography, spacing)
- Interaction notes and motion guidance

## Handoff

- Provide clear references for claude-code to implement.
- If assets are generated, include file paths and usage notes.
